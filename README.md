# Grup Botu

Badini dilinde Telegram grup botu.

## Özellikler
- /test - Bot durumu
- /r - Eksi puanı olanlar
- /id - Kullanıcı bilgileri
- 24 saat konuşmayanları otomatik etiketle
- Günlük en aktif 5 kişi raporu

## Kurulum
1. config.py içinde BOT_TOKEN ve ALLOWED_GROUP_ID'yi ayarla
2. Railway'de Deploy et
